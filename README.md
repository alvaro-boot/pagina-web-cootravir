# Sitio Web Corporativo COOTRAVIR

Sitio web corporativo para COOTRAVIR desarrollado con Vue.js 3 y Vite.

## Características

- Diseño institucional serio y tecnológico
- Tipografía Plus Jakarta Sans
- Paleta de colores COOTRAVIR (azul institucional y dorado)
- Diseño responsive
- Navegación completa con menús desplegables
- Formularios de contacto y cotización con validación

## Tecnologías

- Vue.js 3 (Composition API)
- Vue Router 4
- Vite
- CSS puro (un solo archivo styles.css)

## Instalación

```bash
npm install
```

## Desarrollo

```bash
npm run dev
```

## Construcción para producción

```bash
npm run build
```

## Vista previa de producción

```bash
npm run preview
```

## Estructura del Proyecto

```
pagina web/
├── public/          # Archivos públicos
├── src/
│   ├── components/  # Componentes Vue
│   │   ├── Layout/  # Componentes de layout (Header, Footer, Navigation)
│   │   └── UI/      # Componentes UI reutilizables
│   ├── views/       # Vistas/páginas
│   ├── router/      # Configuración de rutas
│   ├── styles.css   # Estilos globales (único archivo CSS)
│   ├── App.vue      # Componente principal
│   └── main.js      # Punto de entrada
├── package.json
└── vite.config.js
```

## Páginas

- Inicio
- Conócenos (Noticias, Políticas, Responsabilidad Social, Documentos Públicos)
